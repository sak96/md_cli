# Tui

Terminal UI is explained here.

## AppView

```ascii
    +--Folder Name--+--Selected Item---------------+
    | Folder 1/     |                              |
    | Folder 2/     |  <error message if any>      |
    | Folder 3/     |                              |
    | Note 1        |                              |
    | Note 2        |                              |
    | Note 3        |                              |
    |               |                              |
    |               |                              |
    |               |                              |
    +----------------------------------------------+
    |:                                             |
    +----------------------------------------------+
```

## Key Maps

| keys               | action              |
| ------------------ | ------------------- |
| j,k,up,down        | movement in list    |
| l, enter, right    | select item in list |
| h, backspace, left | go back             |
| a                  | add folder/note     |
| d                  | delete folder/note  |
| m                  | move note           |
| c                  | copy note           |
| :                  | interpreter         |
